# Steam Workshop Publishing Checklist

## Required Before Upload

- Replace `About/Preview.png` with the real Workshop preview image.
- Replace `Workshop/icon.png` and `Workshop/cover.png` if you want to use them in the page artwork.
- Build `Assemblies/RimTouch.dll` against the current Steam RimWorld 1.6 assemblies.
- Test with only Harmony and RimTouch enabled.
- Test once with the usual camera/UI mods you want to mention as compatible.
- Keep `About/PublishedFileId.txt` absent for the first upload. RimWorld will create it after publishing.

## Suggested Steam Tags

- 1.6
- Interface
- Utilities
- Quality of Life

## Suggested Load Order

1. Harmony
2. SimpleCameraSetting, optional
3. RimTouch

## Suggested Visibility Flow

1. Upload as hidden or friends-only.
2. Subscribe to your own item.
3. Launch RimWorld from Steam and verify the Workshop copy loads.
4. Check mod settings, UI tap, local map pan, two-finger pinch, world map pan, and double tap.
5. Replace placeholder artwork.
6. Switch visibility to public.
